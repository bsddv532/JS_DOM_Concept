Difference between innerText and innerHTML

innerText :- It will return all text contained by an element and all its child element.

innerHTML :- It will return all text including html tags and that is contained by an element.

//Difference between innerText and innerHTML 
function InnerText(){
    const demo = document.getElementById("task")
    alert(demo.innerText)
}

function InnerHTML(){
    const demo = document.getElementById("task")
    alert(demo.innerHTML)
}
